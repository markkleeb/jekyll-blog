ScreamBooth

For Halloween, we pulled a team together to create a photo booth for our annual NYU party. Using an animated gif library in Processing (thanks to Danne Woo), an HD webcam (thanks to Guilherme Costa), and some killer CSS and web fonts (Yucef Merhi), the ScreamBooth was created.

We set up a small microphone attached to an Arduino. When someone screams loud enough, the Arduino triggers the Processing sketch to snap four quick photos in succession. These photos are then saved to Dropbox and the image IDs are posted to a Python web app. We had a live website that was constantly updating as more pictures were added.

Though optimized for Halloween, the hope is that the Scream Booth can be modified for any holiday or occasion. Look for it at an upcoming party in NYC!
