Fractal Earrings

created by: Mark Kleback

These earrings were fabricated for my girlfriend Stephanie's birthday. The material is mirrored acrylic, and they were etched on a laser cutter.

Design

For the design, I took a fractal tree pattern from my Nature of Code class at ITP and exported the sketch as a PDF file.

Then, I imported the file into Illustrator to create the cuts and etches for the laser.

I tested the pattern on cardboard first, then set up the final cut on the acrylic.
